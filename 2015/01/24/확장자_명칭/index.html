확장자 명칭

2015-01-24
  • 명칭
  • 확장자

우리가 어느 프로그램을 사용할때나 저장할때 꼭 필요한것이 있는데

가령 저장시 제목이나 프로그램 이름뒤에 exe, gif 등을 사용한것을 알수있습니다

이것은 우리가 컴퓨터를 사용할때 쓰는 이름의 스위치로 보면 되겠지요

내가 쓰고 저장하거나 그것을 실행할때는 확장자라는 이름이 있는데

그럼 아래 확장자의 사용용도에 따라 쓸수있는걸 볼까요.

확장자
설명

.apk
Android Package의 약자, 안드로이드 운영체제용 응용 프로그램 설치 파일

.avi
마이크로소프트에서 만든 Audio Video Interleave 동영상 파일 규격

.bat
한 번에 연속적으로 명령어나 응용 프로그램을 실행하게 하는 배치(batch) 파일

.bmp
윈도우에서 주로 사용하는 비트맵(Bitmap) 그림 파일

.com
도스에서 사용하는 명령어(Command) 실행 파일

.cpp
C++ 프로그램 언어에서 사용하는 소스 코드가 담긴 파일

.dat
데이터(Data)가 담긴 파일, 응용 프로그램의 종류에 따라 내용은 달라질 수 있다

.dll
동적 링크 라이브러리(Dynamic-link library)의 약자, 여러 응용 프로그램들이 공통으로 사용하는 함수가 담긴 파일

.doc
MS 워드 프로그램에서 제작한 문서(Document) 파일

.docx
2007년판 이후 버전의 MS 워드에서 제작한 문서 파일

.exe
도스 및 윈도우용 응용프로그램 실행(Executable) 파일

.flac
Free Lossless Audio Codec, 용량을 압축해도 음질 저하가 없는 음성 파일 규격

.flv
플래시(Flash) 규격으로 만들어진 동영상 파일

.gif
Graphics Interchange Format, 256색으로 구성된 그림 파일, 간단한 애니메이션 효과를 넣는 것도 가능

.html
Hyper Text Markup Language, 일반적으로 많이 쓰이는 인터넷 문서 파일의 확장자. ‘htm’으로 줄여 쓰기도 한다

.hwp
아래아 한글 문서 제작기(Hangul Word Processor)에서 제작한 문서 파일

.iso
CD나 DVD에 있는 내용을 추출해 압축한 파일

.jpeg
JPEG(Joint Photographic Experts Group)에서 제정한 사진 파일 규격, ‘jpg’로 줄여 쓰는 경우가 많다

.lnk
윈도우용 바로 가기(단축 아이콘)용 파일

.mid
미디(Music Instrument Digital Interface)형식 전자악기의 악보 코드가 저장된 파일

.mkv
마트로스카(Matroska) 형식으로 저장된 동영상 파일

.mp3
MPEG Audio Layer-3 규격에 의해 압축된 음성 파일

.mp4
MPEG-4 규격으로 압축된 동영상, 혹은 음악 파일

.pdf
Portable Document Format, 어도비 아크로뱃 프로그램으로 제작한 문서

.png
Portable Network Graphics, 휴대용 기기 및 네트워크 환경에 최적화된 그림 파일

.psd
어도비 포토샵(Photoshop)으로 제작한 그림, 혹은 문서(Document)

.pic
매킨토시 컴퓨터용 그림(Picture) 파일, pict나 pic로 표기하기도 한다

.sys
컴퓨터 시스템(System)에 설치된 운영체제 및 응용프로그램의 설정이 기록된 파일

.txt
문자 및 기호로만 이루어진 가장 기본적인 문서(text) 파일

.wav
파형(Wave) 음성 데이터를 압축하지 않은 상태로 저장한 파일

.wma
Windows Media Audio의 약자, 마이크로소프트가 개발한 음성 파일 규격

.zip
단일 파일의 크기를 줄이거나 여러 개의 파일을 하나로 합친 압축 파일의 종류

이상 위 파일 종류나 규격을 나타내는 확장자였습니다.
