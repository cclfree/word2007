MS office word 2007로 티스토리 글쓰기 설정.

2015-01-21
  • 2007로
  • MS
  • office
  • word
  • 글쓰기
  • 설정
  • 티스토리

문서작성에 주로 사용하고 있는 ms word 로 티스토리를 로그인하지않고 원격으로 글쓰는 방법에 대해서 알아보도록 하겠습니다. 이미 2008년도부터 사용이 가능한 기능이지만 아직까지도 사용하는분들이 그리 많치는 않은듯합니다.

문서툴로 글을 작성할경우에는 여러가지 부가적인 기능과 오타를 수정할수가 있습니다. 또한, 여러개의 계정을 사용하고 있다면, 쉽게 관리가 가능하다는 장점을 같고 있는데요.

티스토리에서는 원격으로 포스팅하기위한 기능으로 글 설정 -> BlogAPI 라는 기능을 포함하고 있습니다. 글설정에서 하단에 보시면 워드 프로그램을 설정하는 방법에 대해서도 어느정도 설명이 되어있지만, 좀더 자세히 알아보도록 할까 합니다.

워드를 실행하여 새로만들기 를 선택합니다.

새문서및 최근문서에서 새 블로그 게시물을 선택합니다.

계정등록창이 뜨게되면 지금 등록을 선택합니다.

새 블로그 계정창에서 기타 를 선택하고 다음으로 넘어갑니다.

자신의 블로그계정으로 가서 글 설정 -> blogapl 선택합니다.

API 주소를 드레그하여 복사를 합니다.

다시 워드로 가서 새계정창에서 블로그 게시 URL 을 방금복사한 API 주소를 넣습니다.

사용자 이름과 암호는 본인의 티스토리 로그인정보를 입력하시면 됩니다.

이같이 설정을 해 놓으시면 앞으로 따로 로그인을 하지않고도, MS WORD 프로그램을 실행하여 바로 글쓰기를 하시면 등록을 할수가 있습니다. 블로그의 계정이 여러개가 되신다면 계정관리를 선택하여 추가를 하시면 여러개의 계정을 운영을 할수가 있습니다.~
