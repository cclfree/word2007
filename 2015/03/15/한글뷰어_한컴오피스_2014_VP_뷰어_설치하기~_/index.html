한글뷰어

한컴오피스 2014 VP 뷰어

설치하기~!

여러가지로 편리해지는 세상이려니 하지만, 하드웨어 만큼이나 소프트웨어도 발전을 거듭하는가 봅니다. 아직 한컴 오피스 2007에 익숙한 저로써는 이번 한컴오피스 2014 VP 출시가 있었다는 소식이 참으로 놀랍더군요. 이에 따라서 한컴오피스 소프트웨어의 설치를 하지 못하는 경우에는 한글뷰어를 PC에 설치를 해서 사용할 수 있죠. 한글뷰어는 문서의 작성과 편집에는 제한이 있지만, 단순하게 해당 파일을 불러와서 읽는 데에는 큰 지장이 없답니다. 오늘은 한글뷰어 한컴오피스 2014 VP 뷰어 설치하기에 대해서 알아볼까요?

한글뷰어 한컴오피스 2014 VP 뷰어 다운로드 어디서?

잠시 검색을 해보니 포털에서도 다운로드 링크가 보이지 않더군요. 한컴오피스 2014 VP 뷰어를 받기 위해서는 한컴오피스 홈페이지에 방문하셔야 합니다.

한글과 컴퓨터 바로가기

한글뷰어 다운 및 설치를 위한

한컴오피스 홈페이지 입니다.

2014 버전 출시 안내가 있네요~!

상단 메뉴에 [고객센터] - [다운로드]를

선택합니다.

[다운로드] 페이지에 좌측 메뉴에

보시면 [뷰어] 메뉴창이 있는

요 버튼을 눌러 다음 페이지로 넘어갑니다.

PC OS 맞는 버전을 선택합니다.

저는 윈도우 한컴오피스 2014 VP 뷰어를

선택합니다.

보시면…

한컴오피스 2014 VP 뷰어는

한글/ 한셀/ 한쇼 뷰어를 통합한

버전이고

워드, 엑셀, PPT 문서도

불러와 읽을 수 있습니다.

그런데 다운로드는

H& Friends 홈페이지로 넘기네요~!

한글과 컴퓨터 한글뷰어 한컴오피스 2014 VP 뷰어 다운로드

한컴서포터즈가 H& Friends 로 개편이 되면서 2014 한글뷰어 다운로드는 요기서 받으셔야 합니다. ^^

H& Friends에서 한글뷰어를

다운 받을 수 있는데요..

좌측에 위와 같은 뷰어 다운로드

버튼을 눌러 줍니다.

한컴오피스 2014 VP 뷰어는

윈도우 XP SP3/ VISTA/윈도우 7

/ 윈도우 8.1 설치가 가능합니다.

권장사양은 IE 8.0 이상

CPU 펜티움 4 이상

램은 1GB 이상이어야 합니다.

다운받기 버튼을 눌러줍니다.

보통 개인의 경우 승인이 필요없이

무료로 사용이 가능하지만,

기업의 경우에는 승인이 필요합니다.

하단에 표시된 실행/저장을 눌러

원하는 방법으로 설치 준비를 해줍니다.

저는 실행 버튼을 눌렀습니다.

한컴오피스 2014 VP 뷰어 다운로드를 시작합니다.

한컴오피스 2014 VP 뷰어를

다운 받고 나면 위와 같이

설치 화면으로 넘어가네요~!

다음을 눌러 진행합니다.

사용권 계약서 내용에 동의하고

다음으로 진행합니다.

설치 이전에 연결한

파일의 종류를 설정합니다.

기본으로 한글/ 한셀/ 한쇼가

체크 되어 있는데,

워드/ 엑셀/ PPT를

체크해주면 좋습니다.

설치를 눌러 진행합니다.

설치가 진행이 되고 나면..

한컴오피스 2014 VP 뷰어

설치 종료 창이 나오고

마침을 눌러줍니다.

정상적인 사용을 위해

시스템 재부팅을 권장합니다.
